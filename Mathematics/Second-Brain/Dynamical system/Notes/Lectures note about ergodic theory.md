# Ergodic measures
## Def :: Ergodic

^d8b2bc

The transformation $T$ above is called ergodic if $\forall A\in a$, such that $T^{-1}(A)=A$ then $\mu(A)=0$ or $=1$.

### Rmk :: All subsets that are invariant under the transformation are of trivial measure.
### Rmk :: If $T^{-1}(A)=A$, then $T:A\rightarrow A$, then $T:A\rightarrow A$ and $T:X\backslash A\rightarrow X\backslash A$.

## Thm :: Other equivalent definitions of ergodic
1. $T$ is ergodic;
2. $\forall A\in a: \mu(T^{-1}(A)\Delta A)=0$, then $\mu(A)=0$ or $=1$.
3. For any measurable $f:X\rightarrow \mathbb{R}$, if $f$ is invariant under the transformation $T$, i.e. $f(Tx)=f(x)$ almost everywhere, then $f$ is a constant almost everywhere.


## Def :: Mixing

^3b262f

$T$ is mixing if $\forall A, B\in a: \mu(T^{-n}(A)\bigcap B)\rightarrow B\rightarrow \mu(A)\mu(B)$.

## Thm :: other equivalent definition of mixing
$T$ is mixing if and only if $\forall f,g \in L^{2}(X): \int_{X}f(T^{n}x)g(x)\text{d}\mu(x)\rightarrow\int_{X}f\text{d}\mu\int_{X}g\text{d}\mu$.


The relation between mixing and ergodic is given as 
## Thm :: mixing $\Rightarrow$ erogidc

## Thm :: Conditional expectation
Given a measure space $(X,a,\mu)$ with $\mu(X)=1$, $\mathcal{B}\subset a$ as sub-$\sigma$-algebra, $f:X\rightarrow\mathbb{R}$ a measurable function.

*Question* : Find an approximation for a $a$-measurable function $f$ which is $\mathcal{B}$-measurable?

### Ex :: $X=A_{1}\sqcup...\sqcup A_{l}$ , $A_{i}\in a$.
In this measurable space, for the algebra $a$, there is the sub-algebra $\mathcal{B}$ which is an algebra generated by $A_{i}$. Define $\mathcal{B}$-measurable $f':X\rightarrow \mathbb{R}$ by $$f'(x)=\frac{1}{\mu(A_{i})}\int_{A_{i}}f\text{d}\mu.$$

### Ex :: $L^{2}(X,\mathcal{B},\mu)$ is closed in $L^{2}(X,a,\mu)$. 
Because it 's complete. We have the orthogonal projection: $$P: L^{2}(X,a,\mu)\rightarrow L^{2}(X,\mathcal{B},\mu).$$
For $g\in L^{2}(X,\mathcal{B},\mu)$ and $f\in L^{2}(X, a ,\mu)$, 
$$\int_{X}g\cdot P(f)\text{d}\mu=<\overline{g},P(f)>=\int_{X}g\cdot f\text{d}\mu$$
in which $P(f)=f-(f-P(f))$ and $f-P(f)$ belongs to $L^{2}(X,\mathcal{B},\mu)^{\perp}$.

### Statement of the theorem:
There is a unique linear map $E(\cdot|\mathcal{B}):L^{1}(X,a,\mu)\rightarrow L^{1}(X,\mathcal{B},\mu)$:
1. $E(\cdot|\mathcal{B})$ is $\mathcal{B}$-measurable;
2. For any $g\in L^{\infty}(X,\mathcal{B},\mu)$ with infty norm $||f||_{\infty}=\text{max}|f|$ and $f\in L^{1}(X,a,\mu)$: $$\int_{X}g\cdot E(f|\mathcal{B})\text{d}\mu=\int_{X}g\cdot f\text{d}\mu.$$

where $a_{T}=\{A\in a| \mu(T^{-1}(A)\Delta A)=0\}$.

### Thm :: $L^{1}(X,a_{T},\mu)=\{f\in L^{1}(X,a,\mu)|f(Tx)=f(x)\ \text{almost everywhere}\}$.



# Pointwise ergodic theorem (Birkhoff ergodic theorem)

^7f42be

## Thm :: pointwise ergodic theorem(Birkhoff ergodic theorem)
For any $f\in L^{1}(X)$, then for almost every $x\in X$, $\lim\limits_{N\rightarrow\infty}A_{N}(f)(x)$ exists.

### Rmk :: 
We know that $$A_{N}(f)\rightarrow E(f|a_{T})\ \text{in}\ L^{1}(X).$$
Then $$A_{N}(f)\rightarrow E(f|a_{T})\ \text{almost everywhere},$$
which is alone a subsequence.
So in fact, $$\lim\limits_{N\rightarrow\infty}A_{N}(f)=E(f|a_{T})\ \text{almost everywhere}.$$
### Ideas of proof
1. $\lim\limits_{N\rightarrow\infty}A_{N}(f)(x)$ exists almost everywhere for $f\in W$, $W\subset L^{1}(X)$ a dense subset.
2. Extend convergence to $f\in\overline{W}$.
For the subspace $W$ be the span of:
- $g-g\circ T$ for $g\in L^{\infty}(X)$;
- $g$ for $g(Tx)=g(x)$ almost everywhere, $g\in L^{2}(X)$.

### Thm :: $W$ is dense in $L^{1}(X)$.

### Thm :: $\forall f\in W: \lim\limits_{N\rightarrow\infty}A_{N}(f)$ exists almost everywhere.

### Thm :: Suppose that for $f\in L^{1}(X)$ and $\alpha>0$, denote that $A^{*}f=\text{sup}_{N\geq1}|A_{N}f|,$ then $$\mu(\{x\in X|A^{*}f(x)>\alpha\})\rightarrow 0\ \text{as}\ ||f||_{1}\rightarrow0.$$

### Thm :: Let $\mathcal{L}=\{f\in L^{1}(X)|\lim\limits_{N\rightarrow\infty}A_{N}(f)\ \text{exists almost everywhere}\}$, Then $\mathcal{L}$ is closed in $L^{1}(X)$.


### Thm :: maximal inequality
$\forall f\in L^{1}(X)$ and $\alpha>0$, $$\mu(\{x\in X| A^{*}f(x)>\alpha\})\leq \frac{3}{\alpha}\cdot ||f||_{1}.$$

### Thm :: Covering lemma
Let $\{I_{k}\}_{k\in K}$ be a finite collection of intervals in $\mathbb{Z}_{>0}$.
Then there exists a subcollection $\{I_{k}\}_{k\in K'}$:
1. $I_{K_{1}}\bigcap I_{K_{2}}=\emptyset$ for $k_{1}\not=k_{2}\in K'$;
2. $|\bigcup_{k\in K'}I_{k}|\geq \frac{1}{3}|\bigcup_{k\in K}I_{K}|$.

### Thm :: maximal inequality for $\mathbb{Z}_{>0}$:
$\phi: \mathbb{Z}_{>0}\rightarrow\mathbb{R}$ with the 1-norm $||\phi||_{1}=\sum\limits_{n\in \mathbb{Z}_{>0}}|\phi(n)|$ 
And the time average of function $\phi$: $$A_{n}\phi(x)=\frac{1}{n}\sum\limits^{n-1}_{i=0}\phi(x+i),$$
$$A^{*}\phi=\text{sup}_{n}|A_{n}\phi|.$$
Then $\forall \phi\in L^{1}(\mathbb{Z}_{>0})$ and $\alpha>0$, we have that $$|\{u\in\mathbb{Z}_{>0}\ |\ A^{*}\phi(u)>\alpha||\phi||_{1}\}|\leq\frac{3}{\alpha}.$$
