## Def :: Circle rotation
First one should note the identify in mind:$S^{1}\cong \mathbb{R}/\mathbb{Z}\cong [0,1]$
Circle rotation is defined as $$R_{\alpha}: S^{1}\rightarrow S^{1},$$
in which $\alpha$ is the rotation angle.
